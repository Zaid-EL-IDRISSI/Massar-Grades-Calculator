## Description
This **Python** program allows students to automatically calculate their **overall average** from the grades available on Moroccoâ€™s **Massar portal**. It logs into the student account, fetches the report cards, and calculates both **subject-wise averages** and the **general average**.

---

## Features
- Secure login to the Massar portal  
- Automatic retrieval of student report cards  
- Calculates averages per subject, including activity scores and coefficients  
- Calculates the overall general average
